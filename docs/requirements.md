# 时光轴 — 产品需求文档

## 产品概述

**时光轴** 是一款运行在浏览器中的个人日记分析软件。用户可以自由书写日记，用 `#标签` 标记活动，系统自动识别时间词并关联标签，提供丰富的图表统计分析。

- **运行方式**：浏览器网页应用（PWA，可安装到桌面/主屏幕）
- **目标用户**：个人使用
- **平台**：Windows 电脑 + 手机浏览器（iOS/Android）

---

## 功能需求

### 1. 日记书写

| 功能 | 说明 |
|------|------|
| 自由书写 | 每天一篇日记，支持换行、分段 |
| 标签内嵌 | 在文字中用 `#标签名` 格式，系统自动识别提取 |
| 时间记录 | 独立的起床时间、入睡时间输入框，不在正文里 |
| 时间解析 | 自动识别日记中的时间表达（如"11点"、"下午2点半"），关联到紧跟的标签 |
| 草稿保存 | 自动保存草稿到 localStorage |

### 2. 日历

| 功能 | 说明 |
|------|------|
| 月视图 | 按月显示的日历 |
| 日期标记 | 写过日记的日期有小圆点标记 |
| 日期跳转 | 点击日期跳转到那天的日记 |
| 月份切换 | 左右箭头切换月份 |

### 3. 标签系统

| 功能 | 说明 |
|------|------|
| 新建标签 | 手动添加标签（名称 + 可选颜色） |
| 批量导入 | 一次粘贴多个标签，逗号或换行分隔 |
| 标签颜色 | 每个标签可自定义颜色 |
| 编辑/删除 | 可编辑标签名称和颜色，可删除标签 |
| 预设标签 | 系统自带「起床、睡觉、吃饭、工作、运动、学习、娱乐、其他」 |

### 4. 图表分析

#### 4.1 时间范围
- 今天 / 本周 / 本月 / 年度（四个时间范围切换）

#### 4.2 图表清单

**今天**：
- 24小时时间轴（时间块图）
- 今日标签占比（饼图）

**本周**：
- 24小时时间轴（可选日期）
- 本周标签占比（饼图）
- 本周每日标签对比（竖柱状图）
- 本周睡眠时长（竖柱状图）
- 本周睡眠趋势（折线图）
- 入睡时间分布（饼图）
- 本周起床时间（竖柱状图）
- 本周活动排行（横柱状图，高频在上）
- 本周总结卡片

**本月**：
- 24小时时间轴（可选日期）
- 本月标签占比（饼图）
- 本月标签趋势/4周（竖柱状图）
- 本月睡眠时长趋势（折线图）
- 入睡时间分布（饼图）
- 本月起床时间（竖柱状图）
- 本月活动排行（横柱状图）
- 月度总结卡片

**年度**：
- 年度活跃热力图（日历热力图）
- 年度标签占比（饼图）
- 各月标签趋势（竖柱状图）
- 各月平均睡眠（折线图）
- 年度入睡分布（饼图）
- 各月平均起床（竖柱状图）
- 年度活动排行（横柱状图）
- 年度总结卡片

### 5. 账号与同步

| 功能 | 说明 |
|------|------|
| 注册 | 邮箱 + 密码注册 |
| 登录 | 邮箱 + 密码登录 |
| 多平台同步 | 电脑/手机数据实时同步 |
| 数据加密 | 日记正文客户端加密（AES-256-GCM），统计字段明文 |

### 6. PWA（手机可安装）

| 功能 | 说明 |
|------|------|
| 添加到主屏幕 | 手机浏览器可添加到桌面 |
| 离线基础可用 | Service Worker 缓存 |
| 响应式布局 | 电脑侧边栏导航，手机底部导航 |

---

## 非功能需求

- 数据隐私：日记正文端到端加密，不可被服务器端查看
- 界面语言：中文
- 主色调：白色，简洁直观
- 响应速度：页面加载 < 3 秒
