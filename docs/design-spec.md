# 时光轴 — UI 设计规范

## 设计原则

- **简洁**：减少视觉噪音，让用户专注于日记内容
- **直观**：功能易于发现和使用
- **一致**：统一的颜色、间距、组件风格

## 色彩系统

| 用途 | 颜色 | 色值 |
|------|------|------|
| 页面背景 | 白色 | `#FFFFFF` |
| 侧边栏/卡片背景 | 极浅灰 | `#F9FAFB` |
| 边框/分割线 | 浅灰 | `#E5E7EB` |
| 次要背景 | 浅灰 | `#F3F4F6` |
| 正文文字 | 深灰 | `#1F2937` |
| 次要文字 | 中灰 | `#6B7280` |
| 辅助文字 | 浅灰 | `#9CA3AF` |
| 主强调色 | 蓝色 | `#3B82F6` |
| 主强调色悬停 | 深蓝 | `#2563EB` |
| 成功/起床 | 绿色 | `#10B981` |
| 入睡/夜间 | 靛蓝 | `#6366F1` |
| 警告 | 橙色 | `#F59E0B` |
| 错误 | 红色 | `#EF4444` |

## 字体

- 使用系统默认中文字体栈
- `font-family: -apple-system, BlinkMacSystemFont, "PingFang SC", "Microsoft YaHei", sans-serif`

## 间距

- 基于 4px 网格
- 页面内边距：16px、24px、32px
- 组件间距：8px、12px、16px、24px
- 卡片内边距：16px、24px

## 圆角

- 按钮/输入框：8px (`rounded-lg`)
- 卡片：12px (`rounded-xl`)
- 标签：16px (`rounded-full`)

## 阴影

- 卡片：`shadow-sm`（轻微）
- 下拉菜单/弹窗：`shadow-md`
- 不使用大面积阴影

## 组件规范

### 导航
- 侧边栏（桌面）：220px 宽，白色背景，右分割线
- 底部导航栏（手机）：56px 高，5个图标按钮等分
- 活跃项：蓝色文字 + 浅蓝背景

### 按钮
- 主按钮：蓝色背景 + 白色文字 + 圆角 8px
- 次按钮：白色背景 + 蓝色边框 + 蓝色文字
- 最小高度：36px（触摸目标）

### 输入框
- 白色背景 + 浅灰边框
- 聚焦时蓝色边框
- 圆角 8px

### 卡片
- 白色背景 + 浅阴影 + 圆角 12px
- 图表卡片统一高度

### 标签（tag）
- 圆角全圆 (rounded-full)
- 背景使用标签自定义颜色（浅色版），文字用深色版
- 内边距：2px 10px

### 图表
- 统一使用 Recharts 库
- 饼图颜色循环使用标签自定义颜色
- 柱状图/折线图主色使用蓝色 `#3B82F6`
- 热力图使用绿色系（浅=无活动，深=高活跃）

## 页面结构

### 写日记页
- 顶部：日期显示
- 左侧（桌面）：起床/入睡时间输入
- 右侧（桌面）：日记正文编辑区
- 底部：标签预览（实时高亮显示）

### 日历页
- 顶部：年/月 + 左右箭头
- 主体：7列月视图
- 底部：当天日记预览（卡片）

### 图表页
- 顶部：时间范围切换器（今天/本周/本月/年度）
- 主体：分区排列的图表卡片
- 图表卡片：标题 + 图表 + 图例
