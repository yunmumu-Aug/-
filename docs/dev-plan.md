- [ ] 安装核心依赖（chrono-node, recharts, supabase, lucide-react, date-fns）

- [ ] 设置 Supabase 项目，配置环境变量
- [ ] 配置 CLAUDE.md 指引文件
- [ ] 创建 docs/ 文档文件夹和标准文件
- [ ] 创建 dev-logs/ 开发日志文件夹
- [ ] 部署到 Vercel（空壳验证）

## 开发约定

- 每完成一个阶段，更新 dev-logs 日志
- 保持每个阶段独立可运行，不做超前开发
- 数据库变更通过 Supabase SQL Editor 执行，记录在 changelog
- 敏感配置（Supabase URL/Key）放在 .env.local，不提交到 Git
- Git 提交规范：`feat:` / `fix:` / `docs:` / `chore:` 前缀
