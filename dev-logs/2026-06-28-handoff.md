# 交接文档 — 2026-06-28

## 📋 项目概况

**时光轴（Time Axis）** — 浏览器端日记分析软件，支持多平台同步。
- Next.js 16 + React 19 + TypeScript 5 + Tailwind CSS 4
- Supabase 后端（Auth + Database）
- Recharts 图表
- AES-256-GCM 客户端加密
- PWA 可安装

## 🔧 技术要点

### 标签解析引擎（`tag-parser.ts`）
- 正则匹配 `#标签名` + 时间词（"11点"、"13:00"、"下午2点半"）
- 时间范围支持："13点-14点"、"10点半到11点"
- 距离检查（≤8字符）防止标签"偷走"不相关的时间
- ⚠️ 历史坑：`RANGE_PATTERN` 无 `g` 标志在 `while` 循环中死锁，修复为每次调用新建正则

### 颜色选择器（`tags/page.tsx`）
- 200×200 HSV 色环，`conic-gradient` + `mask` 环形遮罩
- Pointer Events API 拖拽（`setPointerCapture`/`releasePointerCapture`）
- ⚠️ 历史坑：`e.target` vs `e.currentTarget` 导致 `releasePointerCapture` 报错
- 亮度滑块（`bri` state），拖拽使用 `useRef` 读取亮度值防闭包陈旧

### 图表
- Recharts：PieChart（饼图/环形图）、BarChart（柱状图）、LineChart（24h 时间轴）
- 标签点击通过 `sessionStorage` 传递（SVG 渲染无法访问 React hooks）
- 24h 时间轴：睡眠时段蓝色背景，睡眠结束小时不塌缩（边界逻辑修复）

### 手机自适应
- 日历页：桌面大月历 ↔ 手机单周模式（通过媒体查询切换）
- 底部导航（手机）↔ 顶部导航（桌面 md:）
- 日记正文手机铺满（取消左右间距）
- ⚠️ 侧边栏 `sidebar.tsx` 已弃用，保留文件但未引用
