# 开发日志 — 2026-06-28

## ✅ 已完成

### PWA 完善（安装到手机）
用户需求：把"时光轴"安装到手机上使用。

**问题排查过程：**
1. 初始 PWA 缺少 PNG 图标、Service Worker、manifest 不完整
2. 部署在 GitHub Pages 子路径 `/-/` → 所有 PWA 路径需加 basePath 前缀
3. 手机不弹安装提示 → 发现 CDN 版 Workbox 被屏蔽、`beforeinstallprompt` 事件不触发

**最终方案：**
- 自包含 Service Worker（原生 Cache + Fetch API，无外部 CDN 依赖）
- 自定义 build-sw.mjs（不依赖 workbox-build，手写文件扫描+注入）
- InstallPrompt 组件：手机永远显示安装按钮（不依赖事件）
  - Android → 优先用 Chrome 安装弹窗，失败显示菜单指引
  - iOS → 显示 Safari 分享菜单「添加到主屏幕」指引

**技术债：**
- `public/sw.js` 由构建脚本生成，已加入 `.gitignore`
- 构建流水线：`generate-pwa-icons.mjs` → `next build` → `build-sw.mjs`
